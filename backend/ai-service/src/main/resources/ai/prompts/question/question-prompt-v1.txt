다음 조건으로 면접 질문을 %d개 생성하세요.
출력 형식(JSON):
        [
          {
            "title": "string",
            "body": "string",
            "tags": ["string","string"]
          }
        ]
규칙:
        - title은 3~8단어의 짧은 주제명
        - body는 실제 면접 질문 문장
        - tags는 2~4개, 짧은 기술 키워드
        - JSON 외의 텍스트는 절대 출력하지 마세요
조건:
- 면접 질문 분야: %s
- 난이도: %s
- 질문은 실무 중심으로 작성
