<details>
<summary> 수정 사항 (2026.03.30)</summary>

#### 📌 문제 상황
- 같은 사용자가 여러 번 면접 질문 생성 시 ai가 같은 질문을 자주 내려줌

#### 🔍 원인
- 프롬프트 입력에 제외 조건을 넣지 않아 ai가 같은 질문을 내려줌
- 
#### 🛠 해결
- 면접 질문 생성 시 primaryTag를 내려주며 면접 질문 생성 시 
  최근 30개의 질문을 조회 후 primaryTag값을 질문 생성 프롬프트에 추가 하여
  중복 질문 생성 최대한 안되게 설정을 하였습니다.

</details>

- common-spring 모듈에 customcookie 를 생성 후 customcookie를 통해 생성,삭제
  하도록 처리 하였습니다.
